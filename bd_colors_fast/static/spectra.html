Using URL Parameters

The fast spectral explorer can be opened with predefined spectra and display settings by appending query-string parameters.

ParameterValuesEffect
moca_specid, specid, specidsComma-separated spectrum IDsLoads those spectra. Defaults to 13510.
hover1, true, or yesShows detailed hover text for plotted spectral rows.
error_shade1, true, or yesShows a smoothed one-sigma shaded envelope around each displayed spectrum.
snr1, true, or yesDisplays flux divided by flux uncertainty per pixel instead of spectral flux.
xlog, ylog0 disables either log axisBoth axes use logarithmic scaling by default.
fnu_jy, fnu1, true, or yesConverts the display from Fλ to Fν in Jy.
normalize0 disables normalizationNormalizes spectra by the selected wavelength range by default.
normRange in μm, such as 0.95-1.35Sets the normalization range.
hide_ignored0 shows ignored spectral rowsHides data_spectra.ignored = 1 rows by default.
bins, bins_per_micron0 for raw spectra; positive integer for display binningControls server-side wavelength binning. Defaults to raw, unbinned spectra.
showfeatures0 hides featuresShows common chemical and atomic feature bands by default.
disable_lowres1, true, or yesDraws low-resolution spectra as normal lines instead of open markers.
mock1Uses a local synthetic dataset for smoke testing.

Cache

This page keeps downloaded spectra in server memory for a short period. Clearing the cache forces the next load to redownload spectra from MOCAdb.
