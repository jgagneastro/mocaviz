Using URL Parameters

The dual spatial-kinematic explorer accepts the same core URL inputs as the single-view page, but always renders XYZ and UVW together.

ParameterValuesEffect
asso, moca_aid, aidComma-separated association IDsSelects associations. Defaults to HYA,TWA,THA.
mtidComma-separated membership type IDsFilters memberships. Defaults to BF,HM,CM.
oid, moca_oidComma-separated MOCA OIDsOverplots highlighted objects.
bsmdidlatest or a numeric model versionSelects the BANYAN Sigma model version.
showaxes0 or 1Enables the classic 3D plots without background. Defaults to 0.
galaxy0 or 1Controls the ESO Galaxy background in the XYZ panel. Defaults to 1.
checkboxmodels,errors,hover,assmem,likely,subgroups,agecolor,asscenEnables display options in one URL parameter.
mock1Uses a local synthetic dataset for smoke testing.

Cache

The selected membership/model payload is kept in server memory for a short period. Clearing the cache forces the next map load to redownload the selected data.
