Using URL Parameters

This page starts from pcat_gaiadr3_100pc_field for the default field-star sample. Selected associations are added from calc_banyan_sigma with ya_prob >= 90 and use adopted MOCAdb Gaia photometry unless raw Gaia photometry is requested.

ParameterValuesEffect
x1, x2, yGaia band or PSIDSets the color and absolute-magnitude bands. Defaults are GBP, GRP, and G.
asso, moca_aidComma-separated association IDsAdds likely BANYAN Sigma members for the selected associations.
oid, moca_oidComma-separated OIDsOverplots highlighted MOCAdb objects.
ruwe, ruwe_maxPositive number, or noneDe-emphasizes objects with Gaia RUWE above the threshold. Default is 1.4.
color_age1, true, or yesColors objects with BANYAN-linked association ages by age.
raw_gaia1, true, or yesUses Gaia DR3 catalog photometry for selected associations instead of adopted MOCAdb photometry.
extinction_corrected1, true, or yesRequires selected MOCAdb photometry rows to have data_photometry.extinction_corrected = 1.
extinction_vectors1, true, or yesDraws arrows from uncorrected photometry, magnitude + extinction_a, to the corrected plotted photometry when available.
binaries1, true, or yesOverplots known binaries with open symbols. Objects remain plotted normally either way.
sequences0, false, or noHides MOCAdb empirical age-sequence overlays. They are shown by default.
field_hover1, true, or yesEnables hover text for the dense field-star sample. Field hover is off by default.
mock1Uses a local synthetic dataset for smoke testing.

Cache

The selected Gaia CMD payload is kept in server memory for a short period. Clearing the cache forces the next load to redownload the selected data.
