Using URL Parameters

The fast astrometric explorer can be opened with predefined state by appending query-string parameters.

ParameterValuesEffect
moca_oid, oidNumeric MOCA OIDLoads that object. Defaults to 602.
missionsComma-separated mission labelsShows only those mission traces.
subtract_pm1, true, or yesSubtracts the adopted proper motion from the plotted offsets.
subtract_plx1, true, or yesSubtracts the adopted parallax motion from the plotted offsets.
phase1, true, or yesPlots yearly phase instead of absolute epoch.
bin1, true, or yesAdds display-only binned measurements.
adjust_ref0 disables itUses adjusted reference astrometry by default.
only_recalibrated0 disables itUses recalibrated rows, or missions explicitly marked as display-ready, by default.
revert_raw1, true, or yesUses raw, non-recalibrated astrometry when calibration offsets exist.
mock1Uses a local synthetic dataset for smoke testing.
