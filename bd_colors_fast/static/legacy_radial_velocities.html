Using URL Parameters

The legacy radial-velocity page accepts a selected dataset and the usual MOCAdb connection query parameters.

ParameterValuesEffect
datasettarget|template|pipeline_versionLoads one legacy MCMC RV dataset.
target_name, template_name, pipeline_versionExact dataset fieldsAlternative to dataset.
user, pwd, dbase, hostMOCAdb connection valuesOverrides the default connection.
mock1Uses a local synthetic dataset for smoke testing.

Cache

Dataset lists and loaded RV segment payloads are cached in server memory briefly. Clearing the cache forces the next load to query MOCAdb again.
