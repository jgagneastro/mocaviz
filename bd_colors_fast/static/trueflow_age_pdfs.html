Using URL Parameters

The MOCAdb age PDF Explorer accepts the same core target inputs as the legacy Dash page.

ParameterValuesEffect
target, scope, modeobject or associationSelects object-age or association-age products.
oid, moca_oidNumeric MOCA OIDLoads one object's age PDFs.
aid, moca_aidAssociation IDLoads one association's age PDFs.
sourcesMOCAFlows,Legacy,Scalar GaussianControls the initially visible data products.
checkboxposteriors,hbm,log_x,log_y,cdf,combineEnables MOCAFlows and display options.
mock1Uses a local synthetic dataset for smoke testing.

Cache

Loaded age-PDF payloads are kept in server memory briefly. Clearing the cache forces the next target load to redownload and decode the selected products.
