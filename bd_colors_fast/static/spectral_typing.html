Using URL Parameters

The fast spectral typing page can be opened with predefined state by appending query-string parameters to the URL.

Common examples

  • ?specid=602&grid=field&bins=200
  • ?specid=602&grid=field&grid_index=4&norm=0.860-1.350,1.445-1.800,2.010-2.400
  • ?specid=602&deredden=1&fix_rv=3.1
  • ?specid=602&cloud=1
  • ?mock=1&specid=602 for a network-free local sample dataset.

Spectral typing parameters

ParameterValuesEffect
specid, moca_specid Numeric MOCAdb spectrum ID Preselects and computes the comparison spectrum.
grid Spectral grid ID, such as field Selects the standards grid. If omitted, the page starts from the grid with the best finite chi2 value after a comparison is computed.
grid_index Non-negative integer Selects the starting standard within the chosen grid. If omitted, the best chi2 standard in the selected grid is shown.
bins Integer from 1 to 2000 Sets the number of bins per micron used when processing spectra. Default is 200.
norm Comma-separated wavelength windows in microns Sets the normalization regions. Separators like 0.860-1.350,1.445-1.800 are accepted.
deredden 1, true, or yes Fits and applies dereddening to the standards before computing the final displayed match. This is slower.
fix_rv Positive number, or free When dereddening is active, holds R(V) fixed and optimizes only A(V). Use fix_rv=free or leave the field blank to fit R(V).
cloud, cloud_correction 1, true, or yes Fits a one-parameter brown-dwarf slope correction. This cannot be combined with deredden.
cloud_alpha, cloud_alpha_fixed Positive number; cloud_alpha_fixed=0 or cloud_alpha=free to fit Controls the slope-correction wavelength exponent. In the page controls, entering a value fixes alpha; clearing the field fits alpha.

Cache

This page keeps the standards grid, downloaded spectra, and computed comparison payloads in server memory for a short period. Clearing the cache forces the next interaction to redownload and recompute.
