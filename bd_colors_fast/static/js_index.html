MOCAdb JavaScript data visualizations

This page lists the faster JavaScript versions of the MOCAdb visualization tools. Choose one of the following tools:

  • Brown Dwarf Photometry Explorer

    Compare MOCAdb objects against photometric, chemical equivalent width or spectral index sequences versus colors or spectral types with MOCAdb sequence overlays.

  • Gaia CMD Explorer

    Inspect Gaia color-magnitude diagrams for selected associations and individual MOCAdb objects.

  • Spectral Typing tool

    Fit and compare near-infrared spectra against standards, with optional dereddening or slope corrections.

  • Spectral Explorer

    Load, normalize, and compare selected MOCAdb spectra in an interactive Plotly viewer.

  • Astrometric Explorer

    Visualize equatorial-coordinate measurements over time with proper-motion, parallax, reference, and recalibration controls.

  • Spatial-kinematic Explorer

    Explore XYZ positions and UVW velocities for association members, candidate objects, and model ellipsoids.

  • Dual XYZ/UVW Explorer

    Inspect fixed XYZ and UVW 3D views side by side with shared association, object, and model controls.

  • The MOCAdb age PDF Explorer

    Review available age probability density functions including those from MOCAFlows for objects or associations, including summary table metadata.
